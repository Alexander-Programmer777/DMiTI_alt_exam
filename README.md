# DMiTI_alt_exam
Альтернативный экзамен ДМиТИ. 

Тема. Расширение системы компьютерной алгебры-3: Работа над телом невырожденных квадратных
матриц заданного размера n с рациональными
коэффициентами и многочленами
с коэффициентами из этого поля. Реализовать
вычисление значения многочлена от матрицы.

Реализация основной логики работы над телом невырожденных матриц заданного размера n с рациональными коэффициентами и вычисление многочлена от матрицы представлена
в папке Matrixes. Папка состоит из четырех файлов:
1) implementation_choice.py - Реализация разных подходов к одной и той же операции над матрицами (произведение, вычисление определителя, поиск обратной и проч.) с целью исследования 
их производительности и выбора оптимальной. Также в этой папке вы можете увидеть обоснование сложностей алгоритмов и математические подкрепления некоторых реализаций. Код оснащен комментариями с пояснениями некотрых не совсем очевидных моментов.
2) testing.py - Тестирование всех алгоритмов посредством сравнения результатов работы самостоятельно реализованных алгоритмов со встроенными (из библиотеки numpy).
3) research.py - Анализ производительности алгоритмов, имеющих несколько реализаций: построение графиков зависимости скорости выполнения от размера входных данных. Также был написан код (benchmarking.py) в корневой дирректории для запуска в терминале через "Pypy3" (работает значительно быстрее) и удобного сохранения графиков (графики представлены в папке 'graphs').
4) main.py - Файл результата исследования, в котором присутствуют самые оптимальные алгоритмы согласно бенчмаркингу. Содержит итоговые классы Matrix и MatrixPolynomial.

Также на основе финальных классов был улучшен ранее написанный калькулятор. Теперь в нем есть 2 новых раздела: "Матрицы" и "Полином от матрицы".
Запуск калькулятора через файл interface.py.
